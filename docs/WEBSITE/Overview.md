## üìÅ Technical Implementation

This project (`logato-en`) is the **English version** of the Logaa website, built with:

- **Framework:** Next.js (React-based)
- **Architecture:** App Router with route groups (e.g., `(HomePage)`)
- **Components:** Modular component architecture
    - `Header` - Site navigation
    - `Hero` - Landing hero with particle background
    - `AboutSection` - Company introduction
    - `Services` - Four-pillar service cards
    - `WhyLoga` - Value proposition section
    - `ProcessTimeline` - 6-step process visualization
    - `TechStack` - Technology display
    - `FAQAccordion` - FAQ section
    - `CTASection` - Call-to-action with contact info
    - `Footer` - Site footer

### Key Features of the Website

- Scroll-based animations (`useScrollAnimation` hook)
- Particle background effects
- Responsive design
- Modern UI with cards and grids
